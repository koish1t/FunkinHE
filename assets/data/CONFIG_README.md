# songConfig

* this is just for loading the song you want

```json
"songConfig": {
    "songName": "fnf2",
    "difficulty": "hard"
}
```
* that loads fnf2 on hard difficulty

```json
"songConfig": {
    "songName": "bopeebo",
    "difficulty": ""
}
```
* that loads bopeebo on normal difficulty
